# 查找算法

查找算法分类：
1. 静态查找和动态查找；
	- 注：静态或者动态都是针对查找表而言的。动态表指查找表中有删除和插入操作的表。
2. 无序查找和有序查找。
	- 无序查找：被查找数列有序无序均可；
	- 有序查找：被查找数列必须为有序数列。
