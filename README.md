## 如何认知一项技术（中间件、产品等）

**步骤**
- 知道是什么
- 简单使用（实践）
- 原理分析
- 同类对比
- 常见场景处理

**示例**
1. kafka是什么？适用哪些场景？
2. kafka的搭建、基本操作
3. kafka相关的原理（实现原理、集群原理等）
4. kafka同类产品有哪些？各有哪些特点？（rocketmq、activemq、rabbitmq、zeromq等）
5. 如何保证消息的可靠性？如何防止重复消费？消息大量积压如何处理 ？等等场景
