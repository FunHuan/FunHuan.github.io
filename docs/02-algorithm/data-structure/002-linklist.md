# 数据结构：线性+算列 - 跳表

## 链表的基本结构

```
Node<V> {
    Node<V> pointer_1;
    ...
    Node<V> pointer_N;
    V data;
}
```

**特点**
   只能通过链式访问，不能随机访问。

- **单向链表**，只有一个指针，指向下一个节点
  
- **多向链表**，只有多个指针，指向后面的多个节点
  
- **环形链表**，尾部节点指针指向最终指向头部，形成一个环状
